Luas Segitiga

Rumus Luas Segitiga:

L = 0.5 x base x height

- L = Luas Segitiga

- base = alas

- height = tinggi

Luas Lingkaran

Rumus Luas Lingkaran:

L = π x r x r

- L = Luas

- π = Phi

- r = ruas

Luas Jajar Genjang

Rumus Luas Jajar Genjang:

L = base x height

- L = Luas Persegi

- base = alas

- height = tinggi
